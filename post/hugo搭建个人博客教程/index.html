Hugo搭建个人博客教程

本教程适用于Windows端，我电脑的系统是Windows11。如果你使用的是其他操作系统，可以去参考hugo官方文档

1.安装hugo

hugo官方文档 中，推荐使用 Scoop 作为包管理器。

1.1 安装Scoop

安装

安装一般分为三步：配置运行环境，安装Scoop，测试Scoop

配置运行环境

一般要求 powershell 处于可远程执行的状态，所以需要切换运行策略 （ExecutionPolicy）

Set-ExecutionPolicy RemoteSigned -scope CurrentUser

如果需要确认，直接按 Y 确认

安装Scoop

可以参考Gitee作者 RubyKids 的文章 scoop 国内一键安装脚本 (gitee.com)

在PowerShell中运行

iwr -useb https://gitee.com/RubyKids/scoop-cn/raw/main/install.ps1 | iex

测试Scoop

使用 help 命令来测试 Scoop 的状态

scoop help

出现命令日志说明 scoop 安装成功

1.2 安装hugo

scoop install hugo
## 或者
scoop install hugo-extended

安装成功后，测试 hugo 是否安装成功

hugo version

2. 使用 hugo

快速入门： 1. 创建网站 2. 添加内容 3. 配置站点 4. 发布网站

先决条件： 1. 安装 hugo 2. 安装了 git

2.1 创建网站

  • 创建一个项目

    hugo new site myblog

  • 将当前目录更改为项目的根目录

    cd myblog

  • 在项目根目录进行 git 初始化

    git init

2.2 下载并且设置主题

本次演示我们下载 Blackburn |Hugo Themes (gohugo.io) 这个主题

在项目的根目录下

git clone https://github.com/yoshiharuyamashita/blackburn.git themes/blackburn

将一行追加到网站配置文件，指示当前主题。 在根目录的 config.toml 中添加

theme = 'blackburn'

启动 Hugo 的开发服务器以查看站点

hugo server -t blackburn --buildDrafts

2.3 设置文章

  • 新建一篇文章
hugo new post/Hugo搭建个人博客教程.md

  • 修改文章内容

  • 重新启动 Hugo 的开发服务器

hugo server -t blackburn --buildDrafts

3. 将个人博客部署到远端服务器

  • 在Github中建立一个仓库，仓库名为用户名.github.io ，用户名必须是小写

  • 部署

    hugo --theme='blackburn' --baseURL='https://moyujojo.github.io/' --buildDrafts

    然后会在根目录下生成一个 public 文件夹

  • 将 public 文件夹内的东西上传到 github 仓库

    cd .\public\
git init
git add .
git commit -m "我的博客第一次提交" 
git remote add origin git@github.com:Moyujojo/moyujojo.github.io.git
git push # 这里报错The current branch master has no upstream branch.
git push --set-upstream origin master

  • 至此，远端推送成功

4.查看自己的博客

访问网站：My New Hugo Site (moyujojo.github.io)
